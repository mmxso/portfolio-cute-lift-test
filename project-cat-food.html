Project Overview

项目概览

在宠粮品类中，不同价格带面对的人群、购买顾虑和决策因素并不相同。平价用户更关心长期喂养成本，中高端用户更关注成分透明和健康管理，超高端用户则会重视原料等级、体验感和情感价值。

  • 将猫主粮拆分为三个价格带与目标用户群体。
  • 识别从成本、成分信任到高端体验的不同购买顾虑。
  • 定义包装视觉、卖点信息图和页面沟通中的信息优先级。
  • 使用 AI 生成视觉图，将策略想法转化为可放入作品集的表达形式。

Concept Details

方案说明

A

平价 / 大众价格带

大包装更划算，日常喂养无压力

该方案面向多宠家庭、学生党和预算敏感型用户。他们更关注一袋能吃多久、每天花费多少、长期囤货是否划算。因此，视觉表达上采用更醒目的电商信息图风格，用容量对比、每日喂养成本和基础营养图标降低理解成本。

  • 信息优先级：价格 / 容量 / 可喂养天数 / 基础营养
  • 视觉关键词：明亮、直接、热闹、易读
B

中高端 / 科学价格带

把成分讲清楚，让用户有理由相信

该方案面向成分党与理性研究型用户。相比促销感，他们更需要看到配方逻辑与健康利益点。因此，我将肉含量、单一肉源、0谷物、益生菌和营养结构做成清晰的信息模块，帮助用户建立对产品专业度的信任。

  • 信息优先级：成分透明 / 配方逻辑 / 肠胃与毛发健康
  • 视觉关键词：极简、科学、可信、清爽
C

超高端 / 奢华价格带

从宠粮产品，升级为高端喂养体验

该方案面向高投入养宠人群。他们购买的不只是营养价值，也包括新鲜度管理、原料等级和喂养仪式感。因此，视觉重点从普通卖点转向礼盒、小袋分装、冻干工艺和开箱流程，传递更高端的生活方式感。

  • 信息优先级：原料等级 / 新鲜度 / 工艺 / 仪式感
  • 视觉关键词：克制、高端、礼盒感、精致

项目收获

这个项目让我把 AI 图像生成从“做出好看的图片”转化为“验证和表达策略想法”的工具。通过先拆解用户、价格带和购买动机，再生成对应的视觉方案，我能够更清楚地展示自己理解需求、组织信息、判断内容优先级并将分析落地为视觉表达的能力。
